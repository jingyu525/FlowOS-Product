# 新手教程：从零开始掌握你的任务管理系统

欢迎来到你的个人任务管理系统！本教程将通过 5 个简单的步骤，带你熟悉系统的核心功能和工作流。

## 🎯 目标
本系统的核心理念是 **"捕捉一切，专注当下"**。通过将任务从大脑移入系统，你可以减轻焦虑，更专注于重要的事情。

---

## 第一步：认识你的工作台 (The Tour)

首先，让我们熟悉一下系统的目录结构（基于 PARA 方法论）：

- **📂 00-Inbox (收集箱)**: 一切未分类想法的临时存放地。
- **📂 10-Projects (项目)**: 有明确截止日期和目标的任务集合（如“策划年终晚会”）。
- **📂 20-Areas (领域)**: 需要长期维护的责任领域（如“健康”、“财务”）。
- **📂 05-Daily (日记)**: 你的每日工作台，记录当天的流水账。
- **📊 Dashboard (仪表盘)**: 你的指挥中心，概览全局。

---

## 第二步：捕捉你的第一个想法 (Capture)

大脑是用来思考的，不是用来记事的。

1.  打开 **[[00-Inbox/Inbox|00-Inbox/Inbox]]** 文件（如果不存在，请在 `00-Inbox` 文件夹下创建）。
2.  在其中写下你目前脑子里的 3 个待办事项。
3.  **示例**：
    ```markdown
    - [ ] 预约牙医
    - [ ] 买牛奶
    - [ ] 整理电脑桌面
    ```

---

## 第三步：创建你的第一个项目 (Organize)

现在，让我们创建一个正式的项目。

1.  在 **`10-Projects`** 文件夹上右键 -> **New note**。
2.  命名为 `测试项目-学习Obsidian`。
3.  打开新笔记，使用命令 `Cmd/Ctrl + P` -> 输入 `Templater` -> 选择 `Insert template` -> 选择 **[[90-Templates/项目模版|项目模版]]**。
4.  你会看到文件自动填充了结构。在顶部的 `Properties` (YAML) 中，将 `status` 改为 `active`。

---

## 第四步：管理任务 (Engage)

在刚才创建的项目笔记中，添加一些具体任务，并尝试使用 Tasks 插件的语法：

1.  在 "✅ 任务列表" 标题下，输入：
    ```markdown
    - [ ] 阅读新手教程 ⏫ 📅 2026-03-16
    - [ ] 浏览 Dashboard
    ```
    *(注意：`⏫` 代表高优先级，`📅` 后面是截止日期)*

2.  现在，打开 **[[Dashboard]]**。
3.  看！你应该能在 "High Priority" 或 "Today's Focus" 区域看到刚才添加的任务。

---

## 第五步：每日回顾 (Reflect)

每天开始工作前，养成打开日记的习惯。

1.  按 `Cmd/Ctrl + P` -> 输入 `Daily notes` -> 选择 `Open today's daily note`。
2.  系统会自动创建今天的日记文件。
3.  你可以在这里记录会议纪要、临时想法，或者通过 `Tasks` 查询查看今天需要完成的任务。

---

## 🎉 恭喜！

你已经掌握了最基础的工作流：**捕捉 -> 整理 -> 执行**。

- 深入研究 **[[99-Manual/完整用户手册|完整用户手册]]** 了解高级功能。
- 开始把你的真实项目迁移进来吧！
