# 📝 模版库

这里集中管理所有的 Obsidian 模版。你可以通过以下方式使用这些模版：

## 📖 如何使用模版 (How to Use)

> [!help]- 💡 快速指南
> 1. **插入模版**：在任意笔记中，按下 `Cmd/Ctrl + P` 打开命令面板，输入 `Insert template` 并选择要插入的模版。
> 2. **新建模版**：直接在此文件夹 (`90-模版`) 下创建新的 Markdown 文件即可。
> 3. **配置模版文件夹**：请确保在 `设置 -> 核心插件 -> 模版` 中，将 `模版文件夹位置` 设置为 `90-模版`。

## 📂 模版列表 (Template List)

以下是当前库中可用的所有模版文件：

```dataview
TABLE without id file.link as "模版名称", file.ctime as "创建时间", file.mtime as "修改时间"
FROM "90-模版"
WHERE file.name != "模版库"
SORT file.name ASC
```

**提示**: 你可以根据需要修改或添加新的模版。
