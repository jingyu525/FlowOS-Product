# Level 2：进阶指南 - 项目管理 (PARA)

欢迎来到进阶模式！在这个阶段，我们将把散乱的想法组织成有序的**项目 (Projects)** 和 **领域 (Areas)**。

## 🎯 核心理念：PARA 方法论
你的所有任务和资料，都可以归入以下四个类别之一：
1. **P - Projects (项目)**：有明确目标和截止日期的任务集合。
   - *例子*：完成论文、装修房子、策划年会。
2. **A - Areas (领域)**：你需要持续维护的责任范围，没有终点。
   - *例子*：健康、财务、家庭、职业发展。
3. **R - Resources (资源)**：你感兴趣的主题或资料库。
   - *例子*：烹饪食谱、设计素材、读书笔记。
4. **A - Archives (归档)**：已完成的项目或不再活跃的资料。
   - *例子*：2023年已完成的项目。

## 📝 动手练习

### 练习 1：创建一个项目
1. 在 **`10-项目`** 文件夹上右键 -> **New note**。
2. 命名为 `项目-学习FlowOS`。
3. 在新笔记中，输入以下内容（或者使用模版）：
   ```markdown
   # 学习 FlowOS 系统
   状态: #active
   截止日期: 2026-03-31
   
   ## 待办事项
   - [ ] 阅读 Level 1 指南
   - [ ] 阅读 Level 2 指南
   - [ ] 尝试创建一个项目
   ```

### 练习 2：创建一个领域
1. 在 **`20-领域`** 文件夹上右键 -> **New note**。
2. 命名为 `领域-健康`。
3. 在里面记录你的健身计划或体检记录。
