# 📅 日记归档

> **"不经反思的生活不值得一过。"** —— 苏格拉底

## 什么是日记？
这不仅仅是记录“今天吃了什么”的流水账，更是你每天的**工作台**。

**它的作用**：
1.  **记录**：当天的会议纪要、临时想法、工作进展。
2.  **反思**：每天结束时，问问自己“今天做得怎么样？明天有什么计划？”。
3.  **追踪**：通过 Dataview 插件，自动汇总每天的任务完成情况。

---

## 🚀 如何使用？

1.  **日历视图**：查看右侧边栏（Right Sidebar），你会看到一个**日历组件**。点击任意日期即可创建或跳转到当天的日记。
2.  **每日开启**：或者使用快捷键 `Cmd/Ctrl + P` -> 输入 `Daily` -> 选择 `Open today's daily note`。
3.  **调用模版**：系统会自动应用 **[[90-模版/日记模版|日记模版]]**。
4.  **关联项目**：如果在日记中提到某个项目，使用 `[[项目名称]]` 链接到它。

---

## 📊 最近日记 (Last 20 Days)

下方自动展示最近 20 天的日记。你可以在这里快速回顾过去几周的工作。

```dataview
TABLE file.ctime as 创建时间, file.mtime as 修改时间
FROM "05-日记"
WHERE file.name != "日记归档"
SORT file.name DESC
LIMIT 20
```
