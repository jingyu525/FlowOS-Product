# 快速上手指南

这份指南将帮助你快速掌握如何在 Obsidian 中管理你的任务和项目。

## 1. 每日工作流 (Daily Workflow)

每天开始工作时，请遵循以下步骤保持专注：

### 步骤 1：创建每日笔记
1. 使用快捷键 `Cmd/Ctrl + P` 打开命令面板。
2. 输入 `Daily notes: Open today's daily note` 并回车。
3. 如果当天还没有日记，Obsidian 会自动根据 [日记模版](../90-Templates/日记模版.md) 创建一篇新笔记。

### 步骤 2：查看仪表盘
1. 打开 [Dashboard.md](../Dashboard.md) 文件。
2. 浏览 **Today's Focus** 区域，查看今日待办事项。
3. 检查 **Active Projects** 区域，了解当前活跃项目的进度。

### 步骤 3：每日回顾与规划
1. 在每日笔记中，记录当天的想法、会议记录或临时任务。
2. 每天结束时，检查未完成的任务，将需要推迟的任务修改日期，或移动到 `00-Inbox` 中等待重新分配。

---

## 2. 创建新项目 (New Project)

当有一个新的目标或任务集合时，创建一个新项目：

### 步骤 1：创建项目文件
1. 在文件列表中，右键点击 `10-Projects` 文件夹。
2. 选择 **New note**，并为项目命名（例如：`2023-Q4-营销计划`）。

### 步骤 2：应用项目模版
1. 打开新建的项目文件。
2. 使用快捷键 `Cmd/Ctrl + P` 打开命令面板。
3. 输入 `Templater: Insert template` 并选择 [项目模版](../90-Templates/项目模版.md)。

### 步骤 3：填写元数据 (Metadata)
在文件顶部的 YAML 区域（Properties）填写以下信息：
- **status**: 项目状态（如 `active`, `pending`, `completed`）。
- **priority**: 优先级（`high`, `medium`, `low`）。
- **tags**: 相关标签（如 `project`, `marketing`）。

---

## 3. 任务捕捉 (Task Capture)

随时随地记录任务，确保不遗漏任何重要事项。

### 添加任务到 Inbox
如果任务暂时不属于任何特定项目，或者需要稍后整理：
1. 打开 `00-Inbox` 文件夹（如果没有，请在根目录创建）。
2. 在其中创建一个名为 `Inbox.md` 的文件，或者直接在每日笔记中记录。
3. 输入任务，例如：
   ```markdown
   - [ ] 这是一个待办任务 #inbox
   ```

### 添加任务到具体项目
在任何项目文件中，你可以直接添加任务：
```markdown
- [ ] 完成项目初稿 📅 2023-10-27
```

### Tasks 插件语法指南
使用 Tasks 插件的语法来增强任务管理：

- **截止日期**: `📅 YYYY-MM-DD` (例如 `📅 2023-10-27`)
- **开始日期**: `🛫 YYYY-MM-DD`
- **计划日期**: `⏳ YYYY-MM-DD`
- **优先级**:
  - `🔺` 高优先级
  - `🔼` 中优先级
  - `🔽` 低优先级
- **循环任务**: `🔁 every week` (每周重复)

**示例：**
```markdown
- [ ] 每周五提交周报 🔺 📅 2023-10-27 🔁 every week on Friday
```
